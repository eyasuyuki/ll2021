企画についての自由なご意見をお聞かせください

  • 現状程度の規模がよい。
  • 現在の形でよいと思う
  • 安いがいちばん。このぐらいがよい。
  • 妥当な値段なら何日でも
  • フレームワーク対決のようなセッションは複数並行でもよいかも
  • 電源がほしい
  • 時間に余裕がほしいためもっと長い期間を取って開催できるようにしてほしい
  • ストリーミング放送
  • LL飲み会
  • LLライブスクリプティング（期間中ずっとコードを書く）
  • LL分科会（各言語ごとに質疑応答と親睦）
  • LL講習会（体験会：初心者、一見さん大歓迎。LL教えます）
  • LLリレー（プロジェクトの部分を各言語で担当してリレーしていき何かを完成させる）
  • 全国LL（数都市を結んでイベント）
  • LL世界大会（言語開発者のための大会）
  • 高名な方が公開されているアプリ（Webアプリ）をHackする（あるいは修正する）
  • All Nightなどするなら話題の合うどうしが会話しやすい仕組みを考えてほしい
  • Code Jam（Perl Quiz（結城さんの）のようなものを実際にこの場で出題するとか）
  • その場で数名のチームを組んでそれなりに実用的なプログラムを作る
  • MLLC（Multi-Lightweight Language Collaboration）：複数の言語を一緒に使って共同作業（これにより超絶技をあみ出す）
  • Live Hacks（プレゼンではなくひたすらHackしているのを多数のモニタで見るやつ）
  • 温泉はすごくいいですね
  • ぜひ玉造温泉で！
  • 「〜の部屋」をたくさん作り、「フルーツバスケット」といわれたら他の部屋に移らなければならない。
  • 全員参加型Hack
  • 泊まりならMax20人くらいで、ある程度目的を持って（code FESのように）行ったほうがよいと思う
  • 昼過ぎから終電前くらいの時間がいい
  • 友人を誘うためにも入門用のセッションや啓蒙のためのセッションを希望
  • 参加者の男女比が偏っているので、女性はタダにする（男女共同参画委員）
  • 「LL All Nightに参加したい」にしるしをつけたが、現状程度でよいし、しばらくその程度が続くと思っている
  • 「日中のLLに参加したい」にしるしをつけたが、夜でも参加できます
