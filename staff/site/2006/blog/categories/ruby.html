RWikiに脆弱性が発見されました

by takano32 — posted at 2006-05-12 01:20 last modified 2006-07-12 17:38

Rubyで記述されたWikiエンジン，RWikiに任意の命令が実行可能な脆弱性が発見されました．

2.0.1以前のバージョンを利用しているユーザはバージョンアップしましょう．

[ruby-list:42213] RWiki-2.1.0 / RWikiに脆弱性が見つかりました

5月15日に2.1.0pre1から2.1.0のバージョンにXSS脆弱性が発見されました．

当該のバージョンを利用しているユーザはバージョンアップしましょう．

[ruby-list:42218] RWiki-2.1.1 (Re: RWiki-2.1.0 / RWikiに脆弱性が見つかりました)

Rubyist Magazine 0014 号 リリース

by takano32 — posted at 2006-05-16 08:45 last modified 2006-05-16 22:24
　日本Rubyの会有志によるウェブ雑誌、Rubyist Magazine、通称
『るびま』の0014号をリリースしましたのでお知らせします。

http://jp.rubyist.net/magazine/?0014


===========================
= Rubyist Magazine 0014号 =
===========================

http://jp.rubyist.net/magazine/?0014


　『るびま』は、Ruby に関する技術記事はもちろんのこと、
Rubyist へのインタビューやエッセイ、その他をお届けするウェブ
雑誌です。


= Rubyist Magazine 0014号 目次
==============================

* 0014 号 巻頭言
* Ruby の歩き方
* Rubyist Hotlinks 【第 14 回】 角谷信太郎さん
* Ruby ビギナーのための CGI 入門 【第 3 回】 エラーの修正
* RubyOnRails を使ってみる 【第 7 回】 RJS を使ってみる
* qwikWeb の仕組み 【第 3 回】 ページの一部分だけ編集できるようにしてみる
* Miyako の風 ～夢を捨てきれない人たちへ (笑)～
* Rubyist のための他言語探訪 【第 7 回】 Icon
* 0014 号 読者プレゼント
* 求人情報：ザーグソフト
* 0014-RubyNews
* 0014-RubyEventCheck
* 0014 号 編集後記


= 巻頭言
========
書いた人：るびま編集長 高橋征義

　編集長からの 0014 号発行の挨拶です。(難易度：高)


= Ruby の歩き方
===============

　Ruby をはじめるにあたって必要な情報をご紹介します。本稿は
Rubyist Magazine 常設記事です。(難易度：低)


= Rubyist Hotlinks 【第 14 回】 角谷信太郎さん
==============================================

　Rubyist へのインタビュー企画。今回は Ruby 応援団長の角谷さ
んにお話を伺いました。(難易度：いろいろ)


= Ruby ビギナーのための CGI 入門 【第 3 回】 エラーの修正
=========================================================
書いた人：speakillof

　CGI プログラミングを通じて Ruby を学ぶ連載です。たいてい、
CGI プログラミングは Ruby 入門の途中で解説されるものですが、
この連載では CGI プログラミングを通じて Ruby 入門としている
意欲的な企画です。今回はプログラムを書く時には避けられないエ
ラーの修正方法について例を挙げながら解説します。(難易度：簡
単)


= RubyOnRails を使ってみる 【第 7 回】 RJS を使ってみる
=======================================================
書いた人：高木宏 (Gollum)

　master-detail 型の帳票編集を例に、Rails 1.1 で標準となった
RJS を紹介します。(難易度：JavaScript)


= qwikWeb の仕組み 【第 3 回】
  ページの一部分だけ編集できるようにしてみる
============================================
書いた人：えと こういちろう

　Ruby で書かれた Wiki エンジンである qwikWeb の仕組みを解説
する連載です。

　今回は、ページの部分編集について解説します。また、コンテン
ツの性質に応じて編集スタイルを変更する方法についても紹介しま
す。(難易度：206)


= Miyako の風 ～夢を捨てきれない人たちへ (笑)～
===============================================
書いた人：サイロス誠

　Miyako は Ruby で簡単にゲームなどを開発するための環境です。
拡張性がありゲーム以外への応用も可能です。(難易度：中)


= Rubyist のための他言語探訪 【第 7 回】 Icon
=============================================
書いた人：まつもとゆきひろ

　普段 Ruby 使ってるけど、他の言語もいろいろ見てみたい、とい
う人のために、言語オタクとして名高いまつもとさんが Ruby 以外
の言語の紹介をしてくれる連載です。

　今回は Icon です。パッと見るとよくありそうな、よく見るとな
かなか風変わりなような、なかなか個性の強い言語です。(難易度：
検索不能)


= 0014 号 読者プレゼント
========================

　今回もプレゼントがあります。ネットフォレスト様より、Dr.WEB
for Unix のライセンスを 2 つ、arton 様より、『Microsoft .NET
入門 改訂2版』を 2 冊、技術評論社様より『Life Hacks PRESS　
～デジタル世代の「カイゼン」術～』を 3 冊、それぞれご提供い
ただきました。(難易度：発送作業が意外とつらい)


= 求人情報：ザーグソフト
========================

　るびまの新企画、Rubyist への求人情報です。(難易度：名古屋)

2006/08/29

「初心者のためのRuby on Railsチュートリアル」のお知らせ

by takano32 — posted at 2006-08-29 11:45 last modified 2006-08-29 11:46

日本UNIXユーザ会では定期的に勉強会を開催しています．

次回，9月2日の勉強会のテーマは「初心者のためのRuby on Railsチュートリアル」ということで，講師に高橋征義さんをお迎えし，Railsについて講義していただきます．

デモを中心とした，Railsの基本的な使い方についてお話していただく予定です．初心者向けの内容となるため，Railsを使い込んでいる方々には物足りないかもしれませんが，これからRailsを勉強する，LL RingでRailsに興味を持った，という方々はぜひご参加ください．

ご参加くださる場合は詳細の案内をご覧の上，予約システムから事前申込してください．事前申込の締め切りは8月31日となります．よろしくお願いします．
