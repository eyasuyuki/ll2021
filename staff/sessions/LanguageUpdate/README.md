Language Update
====

# セッションの主旨

様々なプログラミング言語の最新動向を素早く把握できるセッションです。今回は以下の言語を取り上げます。(順不同)

- Dart
- Go
- Java
- PHP
- Perl
- Python
- Ruby
- Rust
- Swift

- 開催日: 2021年8月28日(土)
- 開催時間(予定): TBD
